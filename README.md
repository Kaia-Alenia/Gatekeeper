## üìÇ About the Project
This repository contains the **source code logic** for *Gatekeeper*, a psychological horror game and ARG that simulates a realistic Operating System environment.

The goal of this repository is to showcase the **technical implementation** of advanced Ren'Py mechanics, including:
* **Fake OS Interface:** A fully functional desktop environment with windows, taskbar, and file system simulation.
* **"Ghoogle" Search Engine:** Custom search logic (`ghoogle_logic.rpy`) that reacts to player input and lore keywords.
* **Bilingual System:** Dynamic language switching (English/Spanish) with persistent state handling.
* **Threat Mechanics:** Timed events and background logic for "Cleaners" and system intrusions.
